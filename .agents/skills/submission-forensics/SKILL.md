# Submission Forensics

This is a Codex-discoverable bridge to the Storm prompt card at
`../../../skills/submission-forensics.md`.

Before doing a deep audit of another solver's public submission, note, or
promoted diff, read that card and follow its public boundary, source order,
output, and kill gate.

This bridge is local-only. It does not load private chat logs, private
endpoints, raw logs, nonces, telemetry, or always-on behavior.
