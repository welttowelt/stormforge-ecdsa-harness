# Construction Package Gate

This is a Codex-discoverable bridge to the Storm prompt card at
`../../../skills/construction-package-gate.md`.

Before porting or scaling a lower-q construction package, read that card and run
its package gate against the claimed q tier, extra Toffoli, and covered wall
binders.

This bridge is local-only. It does not load private mailbox history, private
endpoints, raw logs, nonces, telemetry, or always-on behavior.
